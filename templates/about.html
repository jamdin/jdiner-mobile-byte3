Accountability: About This Data Set

Dataset Overview

The original dataset contains 13 columns, representing the name of the character, house of allegiance, information about his introduction to the series, information about their death (if they are dead), and information on which book they appeared. In order to obtain the graphs showed, a preprocessing step was necessary and was done using the pandas library of Python on the Google Cloud DataLab. The seaborn library was used for the plots. The preprocessing step consisted of 4 parts:

  • Convert missing values strings to NaN.
  • Create a new column that represents the chapter based on the same referencial timeline, instead of relative to books.
  • Clean the Allegiance column, as some rows contained the prefix "House " and some didn't. It was decided to eliminate this prefix.
  • Convert the labels for Gender from binary to the names "Male" and "Female" for easier analysis.

Where did this data come from?

For this analysis we use the following dataset available on Kaggle that was imported into Google Fusion Tables: https://www.kaggle.com/mylesoneill/game-of-thrones

Who has access to it?

The access is public, and anyone can use the dataset for educational and recreational purposes.
